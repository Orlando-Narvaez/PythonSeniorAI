## 🧠 1. Funciones

### ✔️ ¿Qué es una función?
Una función es un **bloque de código reutilizable** que realiza una tarea específica dentro de un programa.

En lugar de repetir instrucciones varias veces, las funciones permiten **encapsular lógica** y reutilizarla cuando sea necesario.

### ✔️ Ventajas principales
- Evitan la repetición de código.
- Facilitan el mantenimiento del programa.
- Permiten dividir problemas grandes en partes pequeñas.
- Mejoran la legibilidad del código.

---

## 🔧 2. Organización del Código

Una de las ideas centrales de la clase fue entender que los programas bien diseñados deben dividirse en **módulos y funciones claras**.

Esto permite:
- Separar responsabilidades dentro del programa.
- Mantener el código ordenado.
- Facilitar futuras modificaciones.

---

## 🔍 3. Flujo de Ejecución

Cuando un programa utiliza funciones, el flujo de ejecución sigue un proceso:

1. El programa principal inicia.
2. Se llama a una función cuando se necesita realizar una tarea específica.
3. La función ejecuta sus instrucciones.
4. El control regresa al programa principal.

Comprender este flujo es clave para entender programas más grandes.

---

## 🧩 4. Reutilización de Código

Las funciones permiten **reutilizar lógica** en distintas partes del programa.

Esto significa que una misma función puede ser llamada varias veces sin necesidad de reescribir el mismo bloque de instrucciones.

Beneficios:
- Menos errores.
- Programas más cortos.
- Mayor claridad en la estructura.
