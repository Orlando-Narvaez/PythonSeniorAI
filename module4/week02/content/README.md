## 🧠 1. Funciones en Python

Las funciones siguen siendo una de las herramientas más importantes para organizar programas.

Permiten:
- Dividir problemas grandes en tareas pequeñas.
- Reutilizar lógica.
- Hacer el código más claro y mantenible.

En esta semana se profundizó en cómo **utilizar funciones de forma más flexible**.

---

## 🔧 2. Parámetros y Argumentos

Las funciones pueden recibir información desde el exterior mediante **parámetros**.

Estos parámetros permiten que una misma función pueda trabajar con distintos datos sin necesidad de cambiar su lógica interna.

### Importancia
- Hace que las funciones sean reutilizables.
- Permite adaptar la función a diferentes situaciones.
- Facilita la creación de programas más dinámicos.

---

## 🔍 3. Flujo de Datos en Funciones

Durante la clase se analizó cómo los datos:

1. Entran a una función.
2. Son procesados dentro de ella.
3. Generan un resultado.

Comprender este flujo es clave para diseñar programas más complejos.

## 🧩 4. Uso Estratégico de Funciones

Se discutió cómo diseñar funciones correctamente:

- Cada función debe tener **una responsabilidad clara**.
- Las funciones deben ser **pequeñas y específicas**.
- Evitar funciones demasiado largas o con demasiadas tareas.

Este enfoque facilita la lectura y mantenimiento del código.
